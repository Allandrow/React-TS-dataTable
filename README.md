# React-TS-dataTable

A simple headless library to create tables amd components that interact with them.

Built With :

- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)

## Installation

- Install the package in your repo

```shell
  npm i react-ts-datatable
```
